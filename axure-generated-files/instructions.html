REPEATERS

Repeaters are useful because you can type in a dataset or copy and paste from a spreadsheet. To use a repeater, drag it out of the masters panel. To edit the dataset, double click into the repeater, make sure your page properties are visible in View > Panels > Page Properties and click into the left tab marked Repeater Dataset.

 

Repeaters function by taking a template (the widgets you see when you click into it) and applying the dataset to the template. The length of the repeater is set by the number of items in the left column. The text on each widget is set by the PageName column. The other columns show icons. More on this in the ICONS section below.

 

Horizontal Subnav Repeater functions identically to the vertical, and will snap into the columns. This is useful for subpages, where you can just copy the dataset over. Please note that all of the elements in the horizontal repeater are all the same color, and there isn't yet a way to link the datasets between a vertical and horizontal repeater.

GRIDS

The grids in this file are weird looking, but are set up to lock the repeater, flow lines, and flow shapes together.

 

The repeater snaps to the outside guides. The middle guide (The second of the triple line set) snaps the flow line shape so it connects accurately to the repeater. The inner most guide (The third of the triple line set) is the left edge of the flow shape, with the right edge of it snapping to the lone line.
